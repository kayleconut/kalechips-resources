Header

This is the base code for my UTAU website. It's designed to look vaguely like a terminal window. It can be entirely text based if you so choose, or you can add images. The palette is very limited, which makes it pretty fast to set up.

You can copy the source code for this from Github! Or just take it from the view source page directly.

This is a compact layout and is thus best suited for compact websites. I advise limiting the links in the menu to 4 to avoid overflow on smaller phones, unless you want to fiddle with the font sizes to squeeze more in.

If you'd like the window icons for the top, download the image, upload it to your website, and put the appropriate path into the CSS.

As a bonus, the CSS has automatic dark/light mode styling already!
